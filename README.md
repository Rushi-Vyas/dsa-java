# DSA in Java

Practice problems and implementations of Data Structures and Algorithms using Java.

## Topics Covered

- Arrays (sorting, searching, reversal)
- Strings (palindrome, anagram)
- Linked List (singly linked list)
- Stacks (array-based implementation)
- Searching (linear, binary)

## How to Run

1. Clone this repository
   ```
   git clone https://github.com/Rushi-Vyas/dsa-java.git
   ```
2. Open any `.java` file in VS Code or IntelliJ
3. Compile and run:
   ```
   javac filename.java
   java filename
   ```
